
Dashboard de Controle de Inadimplência – Cantina Escolar

Este projeto apresenta uma planilha em Excel desenvolvida para
controlar pagamentos da cantina de uma escola particular.

Funcionalidades:
• Controle de pagamentos
• Identificação automática de inadimplentes
• Cálculo de dias em atraso
• Mala direta para cobrança via WhatsApp
• Dashboard financeiro

Ferramentas utilizadas:
Excel
Tabela dinâmica
Dashboards

Indicadores analisados:
• Faturamento total
• Total pago
• Total inadimplente
• Quantidade de alunos em atraso
