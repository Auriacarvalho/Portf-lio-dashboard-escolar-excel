Problema
Cantinas escolares precisam controlar pagamentos e identificar alunos inadimplentes.

Solução
Desenvolvimento de uma planilha em Excel com controle financeiro e dashboard.

Indicadores analisados
• faturamento total
• valor pago
• valor em atraso
• quantidade de inadimplentes
