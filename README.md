📊 Dashboard de Gestão Escolar e Controle de Inadimplência
📝 Descrição do Projeto
Este projeto foi desenvolvido para solucionar um dos maiores gargalos administrativos de instituições de ensino: a gestão de fluxo de caixa e o controle de inadimplência.

O dashboard consolida dados de matrículas, mensalidades e status de pagamento, permitindo que a gestão escolar identifique rapidamente quais turmas ou períodos apresentam maior índice de atrasos, facilitando a tomada de decisão para cobranças e previsibilidade financeira.

🚀 Principais Funcionalidades
Visão Geral de Receita: Monitoramento em tempo real do faturamento previsto vs. realizado.

Indicador de Inadimplência: Cálculo automático da taxa de atraso por mês e por turma.

Filtros Dinâmicos (Slicers): Navegação intuitiva por ano letivo, nível de ensino e status do aluno.

Destaque de Alertas: Formatação condicional para sinalizar pagamentos críticos.

🛠️ Competências Técnicas Aplicadas
Para a construção deste dashboard, utilizei recursos avançados do Microsoft Excel que garantem a integridade e a escalabilidade dos dados:

Modelagem de Dados: Estruturação de bases de dados utilizando Tabelas Oficiais para garantir atualização dinâmica.

Fórmulas Avançadas: Uso de PROCX, SOMASES, CONT.SES e funções lógicas (SE, E, OU) para tratamento de dados.

Power Query (ETL): Limpeza e padronização dos dados de entrada para evitar erros de duplicidade ou formatação.

Tabelas Dinâmicas: Consolidação ágil de grandes volumes de dados para suporte aos gráficos.

Dashboard Design (UI/UX): Aplicação de princípios de visualização de dados (DataViz) para criar uma interface limpa e focada no usuário final.

📂 Como Utilizar
Faça o download do arquivo .xlsx disponível neste repositório.

Insira os dados de novos alunos na aba Base_Dados.

Clique em Dados > Atualizar Tudo para refletir as mudanças no Dashboard.
